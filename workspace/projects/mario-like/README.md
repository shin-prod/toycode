# mario-like (オリジナル横スクロールアクション)

※スーパーマリオそのもの（キャラ/画像/音）は著作権があるため使用せず、**マリオ風**の挙動を持つオリジナル簡易ゲームとして実装しています。

## 起動方法（ブラウザ）
1. `projects/mario-like/scr/index.html` をブラウザで開く
   - うまく動かない場合は簡易サーバを起動:

```bash
cd projects/mario-like/scr
python3 -m http.server 8000
```

2. ブラウザで `http://localhost:8000` を開く

## 操作
- ← / → : 移動
- Space / ↑ : ジャンプ
- R : リスタート

## ルール
- コインを集めてスコアを上げる
- 敵（スライム）に横から当たるとダメージ（ライフ減）
- 上から踏むと敵を倒せる
- 右端の旗に触れるとクリア

## カスタム
- `game.js` の `LEVEL` 配列をいじると地形・コイン・敵・旗を変更できます。
