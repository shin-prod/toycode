# PowerPoint 作成ルール（厳守）

## 作業手順（必ずこの順序で実行すること）

PPT を作成・編集するときは、以下のステップを**必ず順番通りに**実行すること。
スキップは禁止。

### Step 0: 構成案をユーザーに提示して承認を得る（ツール実行前に必須）

ファイル作成・要素追加など**いかなるツール呼び出しも行う前に**、
以下の形式でスライド構成案をテキストで提示し、ユーザーの確認を取ること。

```
【スライド構成案】

スライド 1: タイトルスライド
  - タイトル: 〇〇〇〇
  - サブタイトル: 〇〇〇〇

スライド 2: 〇〇〇〇
  - レイアウト: タイトル + 本文
  - 内容: 〇〇〇〇

...

この構成でよろしいですか？
```

ユーザーから「OK」「はい」「進めて」などの承認を得てから Step 1 に進むこと。
承認なしでツールを呼び出してはならない。

### Step 1: ファイルとスライドを作成する
- `pptx_add_slide` でファイルを作成する
- 保存先: `~/workspace/assets/ppt/` フォルダ

### Step 2: スライドサイズを必ず確認する
- `pptx_read` を呼び出してスライドの `slide_width_inches` / `slide_height_inches` を確認する
- **この確認を省略してはならない**

### Step 3: 各要素の座標を計算してから配置する

要素を追加する前に、以下を頭の中で計算して確認すること：

```
left + width  ≤ slide_width_inches  (デフォルト 10)   ← 超えたら配置しない
top  + height ≤ slide_height_inches (デフォルト 7.5)   ← 超えたら配置しない
```

境界を超える場合は **width / height を小さくするか、left / top を調整**してから配置する。

### Step 4: 要素を追加する
- テキストボックス: `pptx_add_textbox`
- 図形: `pptx_add_shape`
- 表: `pptx_add_table`
- 画像: `pptx_add_picture`

---

## スライド座標系

```
(left=0, top=0) ─────────────── (left=10, top=0)
       │                                  │
       │     スライド全体                  │
       │     幅 10 インチ × 高さ 7.5 インチ │
       │                                  │
(left=0, top=7.5) ──────────── (left=10, top=7.5)
```

- **幅 10 インチ、高さ 7.5 インチ**がデフォルト
- 端から **0.4 インチ以上**の余白を必ず確保すること

---

## フォントサイズ（この値を守ること）

| 用途 | サイズ |
|------|--------|
| スライドタイトル | 32 pt |
| セクション見出し | 22 pt |
| 本文・箇条書き | 16 pt |
| 図形内ラベル | 12 pt |
| 注釈・補足 | 10 pt |

- `font_size` を省略してはならない。必ず明示すること
- 1行あたり**全角 18〜22 文字以内**に収めること。超える場合は `\n` で改行する

## 配置テンプレート（そのまま使うこと）

### タイトル + 本文（基本構成）
```
pptx_add_textbox: left=0.5, top=0.4, width=9.0, height=1.2, font_size=32
pptx_add_textbox: left=0.5, top=1.9, width=9.0, height=5.0, font_size=16
```

### 左右2カラム
```
左: left=0.4, top=1.8, width=4.4, height=5.0
右: left=5.2, top=1.8, width=4.4, height=5.0
```
※ 4.4 + 5.2 = 9.6 ≤ 10 ✓、5.0 + 1.8 = 6.8 ≤ 7.5 ✓

### 図形3列（横並び）
```
列1: left=0.4,  top=2.0, width=2.8, height=2.5
列2: left=3.6,  top=2.0, width=2.8, height=2.5
列3: left=6.8,  top=2.0, width=2.8, height=2.5
```
※ 6.8 + 2.8 = 9.6 ≤ 10 ✓、2.0 + 2.5 = 4.5 ≤ 7.5 ✓

## よくある失敗と対処

| 失敗 | 原因 | 対処 |
|------|------|------|
| テキストがはみ出す | font_size が大きすぎる | 12〜14 pt に下げる |
| 要素がスライド外にはみ出す | left+width > 10 または top+height > 7.5 | 座標・サイズを見直す |
| 図形が重なる | 座標を確認していない | pptx_read で既存要素を確認してから追加 |
| 文字が詰まって読めない | 1行が長すぎる | \n で 20 文字以内に分割する |
